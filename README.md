# Madinah Dawah Graduates — App Preview

A clickable React preview for the Madinah Dawah Graduates platform.

## Run

```bash
cd madinah-dawah-app
npm install
npm run dev
```

Open http://localhost:5173

## Screens

- `/sponsor` — Sponsor dashboard (default)
- `/graduate/musa-mohsin` — Graduate profile
- `/report/2026-04-14` — Daily report
- `/admin` — Admin dashboard
- `/graduates` — Public browse page

Theme toggle (sun/moon) persists via localStorage.
